Installation du profil iPhone

Télécharge le profil de configuration pour continuer.

Télécharger le profil .mobileconfig

Tutoriel d’installation

  1. Appuie sur Télécharger le profil.
  2. Autorise le téléchargement si iPhone te le demande.
  3. Ouvre l’app Réglages.
  4. Appuie sur Profil téléchargé.
  5. Appuie sur Installer en haut à droite.
  6. Entre ton code iPhone si demandé.
  7. Confirme encore avec Installer.
  8. Une fois terminé, reviens dans la mini app.

Important : installe uniquement un profil provenant d’une source de confiance.
